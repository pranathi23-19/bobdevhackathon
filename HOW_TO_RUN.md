# 🚀 How to Run - Simple Steps

## ⚡ FASTEST WAY (Windows)

### Step 1: Open File Explorer
Press `Windows Key + E`

### Step 2: Navigate to Project
Go to: `c:\Ibmbobhackathon\Bobdevhackathon`

### Step 3: Double-Click to Run
Double-click on: **`start-ai-demo.bat`**

✅ That's it! The browser will open automatically with the AI dashboard.

---

## 📝 Alternative Method (Command Line)

### Step 1: Open Command Prompt
Press `Windows Key + R`, type `cmd`, press Enter

### Step 2: Navigate to Project
```cmd
cd c:\Ibmbobhackathon\Bobdevhackathon
```

### Step 3: Start the Application
```cmd
npm start
```

### Step 4: Open Browser
Go to: `http://localhost:3000/ai-dashboard.html`

---

## ❓ Troubleshooting

### Problem: "Port 3000 already in use"
**Solution**: Another instance is running. Close it first or use a different port.

### Problem: "npm is not recognized"
**Solution**: Install Node.js from https://nodejs.org/

### Problem: Browser doesn't open automatically
**Solution**: Manually open: `http://localhost:3000/ai-dashboard.html`
